# Review Redirect for WooCommerce

Review Redirect for WooCommerce helps WooCommerce store owners collect post-purchase customer sentiment and invite happy customers to leave a Google review.

Customers can select a star rating on the WooCommerce order received page. Ratings at or above the configured threshold reveal a Google review button. Lower ratings show a private feedback form so the store can review issues internally.

## Features

- Review prompt on the WooCommerce order received page.
- Configurable Google review URL.
- Custom popup title and message.
- Positive rating threshold setting.
- Private low-rating feedback saved to WooCommerce order meta.
- Admin email notification for low-rating feedback.
- Optional review request content in completed order emails.
- Order key verification for feedback submissions.
- Per-order prompt suppression in the customer's browser.

## Requirements

- WordPress 6.0 or later.
- WooCommerce 7.0 or later.
- PHP 7.4 or later.

## Installation

1. Upload the `review-redirect-for-woocommerce` folder to `/wp-content/plugins/`.
2. Activate **Review Redirect for WooCommerce** from the WordPress Plugins screen.
3. Go to **Marketing > Review Redirect**.
4. Enable the plugin and enter an approved Google review URL.
